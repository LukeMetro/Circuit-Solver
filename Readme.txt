This program is a circuit solver designed to solve resistor circuits and calculate
solution data for the circuit, namely:
	-Voltage across and current through each resistor
	-Thevenin equivalent resistance of circuit.

This project utilizes the Python Imaging Library to display semi-transparent 
.PNG files. It can be obtained from http://www.lfd.uci.edu/~gohlke/pythonlibs/

Run this program by running StartScreen.py.
